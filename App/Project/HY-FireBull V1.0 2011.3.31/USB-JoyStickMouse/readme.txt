1、该例程为USB-JoyStickMouse例程。

2、使用说明
   (1)工程文件路径：例程目录\USB-JoyStickMouse\MDK-ARM\Project.uvproj。
   (2)请使用MDK 4.0以上版本打开,MDK版本过低会导致无法识别工程。
   (3)下载调试工具为ULINK。
   (4)请将USB线插板子上,JP9插1-2位置。
   (4)HEX文件下载到板子后,LED灯闪烁,电脑识别USB人体输入设备,当按下Joystick(鼠标方向键)、User Button(鼠标右键)、Tamper Button(鼠标左键)后,
      鼠标指针移动到目的地,表明例程运行正确。
	   	
3、注意事项
   请务必在下载、调试、运行过程中,保持板子上电、ULINK连接并插在电脑上。
